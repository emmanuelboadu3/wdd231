Beginner Coding Tutorials Website - Site Plan

Site Name

CodeStart Tutorials – A website designed to teach beginners HTML, CSS, and JavaScript in a simple and interactive way.

Optional domain: codestart.org

Site Purpose

The site provides beginner-friendly coding tutorials, interactive practice exercises, and resources to help users gain confidence in web development. It will cover HTML, CSS, and JavaScript basics, with step-by-step examples and practice challenges.

Scenarios

  • “How do I create a simple webpage using HTML and CSS?”
  • “Where can I practice JavaScript exercises online?”
  • “How do I combine HTML, CSS, and JavaScript in a single project?”

Color Scheme

  • Primary Color: #2C3E50 – for headings and major sections.
  • Accent Color: #D35400 – for links, highlights, and interactive elements.
  • Background Color: #f8f9fa – for page background.

Typography

  • Headings: Playfair Display
  • Body Text: Roboto
  • Code Snippets: Courier New

Wireframe

Mobile View Wireframe

Represents stacked layout, 1-column tutorials and challenges.

Desktop/Tablet View Wireframe

Includes 3-column tutorial grid and 2-column practice challenges.

CSS Plan

Supporting CSS will be created to apply the selected color scheme and typography. Media queries will ensure responsive layouts across mobile, tablet, and desktop.

Testing

Testing will include:

  • HTML Validation: W3C Markup Validator
  • Accessibility: WAVE Web Accessibility Tool
  • Performance & Best Practices: Chrome DevTools Lighthouse
  • Color Contrast: WebAIM Contrast Checker
