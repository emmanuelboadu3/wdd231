Beginner Coding Tutorials Website - Site Plan

Site Name

CodeStart Tutorials – A website designed to teach beginners HTML, CSS, and JavaScript in a simple and interactive way.

Optional domain: codestart.org

Site Purpose

The site provides beginner-friendly coding tutorials, interactive practice exercises, and resources to help users gain confidence in web development. It will cover HTML, CSS, and JavaScript basics, with step-by-step examples and practice challenges.

Scenarios

  • “How do I create a simple webpage using HTML and CSS?”
  • “Where can I practice JavaScript exercises online?”
  • “How do I combine HTML, CSS, and JavaScript in a single project?”

Color Scheme

  • Primary Color: #2C3E50 – used for headings, navigation bar, and main titles.
  • Accent Color: #E67E22 – used for links, buttons, highlights, and interactive elements.
  • Background Color: #f8f9fa – for page background.

Typography

  • Headings: Poppins – clean, readable for titles and section headers.
  • Body Text: Roboto – used for paragraphs and tutorial content.
  • Code Snippets: Courier New – for inline and block code examples.

Wireframe

Mobile View

HEADER: Logo + Hamburger Menu
INTRO SECTION: Welcome message + overview
FEATURED TUTORIALS: 1-column list
PRACTICE CHALLENGES: single column with interactive feedback
FOOTER: Contact info, links, and attribution

Desktop/Tablet View

HEADER: Logo + Horizontal Nav Links
INTRO SECTION: Welcome message + overview
FEATURED TUTORIALS: 3-column grid with images/examples
PRACTICE CHALLENGES: 2-column layout with interactive challenges
FOOTER: Contact info, links, and attribution

CSS Plan

Supporting CSS will be created to apply the selected color scheme and typography. Layouts will be responsive using media queries for mobile, tablet, and desktop. Styling will remain lightweight to optimize performance.

Testing

Testing will include:

  • HTML Validation: W3C Markup Validator
  • Accessibility: WAVE Web Accessibility Tool
  • Performance & Best Practices: Chrome DevTools Lighthouse
  • Color Contrast: WebAIM Contrast Checker
