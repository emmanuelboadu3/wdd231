About CodeStart Tutorials

Purpose

CodeStart is designed to teach beginners HTML, CSS, and JavaScript with clear explanations, interactive exercises, and accessible patterns.

Accessibility

  • Semantic headings and landmarks (header, nav, main, footer).
  • Keyboard-friendly navigation and modals (Escape closes, focus managed).
  • Color contrast meets WCAG AA; visible focus states.
  • Images use alt text and intrinsic size to avoid layout shift.

Performance budget

  • Each page loads under 500 kB from an empty cache.
  • Images compressed and lazy-loaded; JS deferred; CSS minimal.
  • No frameworks; ES modules; cache-friendly assets.
